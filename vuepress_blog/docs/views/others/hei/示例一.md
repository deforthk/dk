## 伟大作家的诞生



::: tip 第一回 <br>
:::
```
                                                        《无题》
                    近御台，杯酒不解愁上愁。乱离人，虚里幽香更柔。若执子之手，末代千秋，人生几何！
                    满晨星，相思难及月如钩。长夜里，情似江水难收。若与子偕老，暮暮朝朝，可有他求？
```


::: tip 第二回 <br>
:::

```
                                                        《无题》
                    莫相送，
                    莫相留，
                    一湾浓月满杯酒。
                    -------------
                    莫相忘，
                    莫回首，
                    漠然残阳已是秋天！ 
                   
```

                        欲把银河添做酒，此生如梦，醉卧天地间！
                        -----------------
                        寒风卷沙起，肃杀眼前空杯人。
